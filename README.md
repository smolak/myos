# MyOS

Local-first productivity dashboard built with Electrobun (TypeScript, Bun, and the native OS webview).

Full design, contracts, and structure are documented in [specs/ARCHITECTURE.md](specs/ARCHITECTURE.md).

## Getting started

```bash
bun install
bun run dev
```

Development with Vite HMR (recommended while working on the shell UI):

```bash
bun run dev:hmr
```
